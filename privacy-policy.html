Privacy Policy

Claude Auto-Continue Browser Extension · Last updated March 17, 2026

Overview

Claude Auto-Continue is a browser extension that automatically clicks the "Continue" button on claude.ai when Claude reaches its per-turn tool-use limit. This privacy policy explains what data the extension does and does not access.

Data Collection

Claude Auto-Continue does not collect, transmit, store, or share any personal data, browsing history, or analytics. There are no servers, no tracking, and no third-party services.

Local Storage

The extension stores a small number of user preferences locally on your device using Chrome's storage.local API. These include:

  • Whether the extension is paused or active
  • The maximum continuation limit you have set
  • The current continuation count for the session
  • Whether the "minimize tokens" option is enabled

This data never leaves your browser and is not accessible to any external service.

Permissions

The extension requests the following permissions, all scoped exclusively to claude.ai:

  • scripting – Inject detection and click scripts into claude.ai tabs
  • tabs – Query open claude.ai tabs across all windows so the extension works in background tabs
  • storage – Persist your settings locally across browser sessions
  • alarms – Keep the background service worker alive so polling continues reliably
  • activeTab – Allow the popup to communicate with the current tab

The extension's host permissions are limited to https://claude.ai/*. It cannot access any other website.

Third-Party Services

The extension does not communicate with any external server. No data is sent to Anthropic, the extension developer, or any third party.

Changes to This Policy

If this policy is updated, the changes will be posted on this page with a revised date. Significant changes will also be noted in the extension's changelog.
